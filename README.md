# 🚀 100 Days of Problem Solving

This repository is my personal journey to becoming better at problem solving and Data Structures & Algorithms.

## 📌 Goal
- Solve 1 problem every day for 100 days
- Start from easy problems and gradually move to harder ones
- Focus on understanding, not just solving

## 🧠 What I Include in Each Problem
- Clear explanation of the problem
- Multiple approaches (Brute Force → Optimized)
- Time and Space Complexity
- Comparison of different methods
- My personal learning notes
- A small practice task for others

## 🎯 Purpose
This repo is not just about solving problems, but about:
- Improving logical thinking
- Learning different ways to solve the same problem
- Helping others learn in a simple way

## 📅 Structure
Each day contains:
- `problem.md` → Explanation + approaches
- `solution.py` → Code implementation
